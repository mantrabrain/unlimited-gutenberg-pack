== Description ==

<strong>Gutenberg Pack - Unlimited Gutenberg Block</strong>

Gutenberg Pack is a addon for gutenberg editor. It extends gutenberg editor to next level. We are developing many other beautiful and handy gutenberg blocks. You will see in next update.


<strong>Gutenberg Pack - Blocks and Descriptions</strong>

<ul>
 	<li><a href="https://gutenbergpack.com/#" target="_blank">Testimonial </a>: This block help you to show some amazing feedback from your clients.</li>
 	<li><a href="https://gutenbergpack.com/#" target="_blank">Buttons </a>: This block help you to show any button with link.</li>
 	<li><a href="https://gutenbergpack.com/#" target="_blank">Tabbed-content</a>: This block help you to organize your content with Tab.</li>
 	<li><a href="https://gutenbergpack.com/#" target="_blank">Call To Action</a>: This block help you to show call to action section on your website.</li>
 	<li><a href="https://gutenbergpack.com/#" target="_blank">Featured</a>: This block help you to show featured section on your website.</li>

</ul>


== Installation ==

1. Install Gutenberg Pack either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.
